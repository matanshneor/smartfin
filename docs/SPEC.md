## מודל הנתונים

- **עסקה:** סכום, סוג (הוצאה/הכנסה/חיסכון), קטגוריה, תיאור ("מהות"), תאריך, בעלים, דגל קבועה.
- **שיוך לבן משפחה (`user_id`):** הוצאות והכנסות — מתן / אור / משותפת (NULL). חיסכון — תמיד משפחתי (NULL), "חוסכים ביחד".
- **קטגוריות:** לכל משפחה משלה (ניתנות לעריכה מלאה). הכנסות = בדיוק 2: משכורת 💼, הכנסה נוספת 💵.
- **עסקאות קבועות:** עסקה מסומנת כקבועה היא תבנית; מנוע `materialize_recurring` יוצר מופעים אמיתיים (מקושרים ב-`recurring_parent_id`) רטרואקטיבית ועד היום — רץ בהוספה/עריכה ופעם ביום בכניסה. מחיקת תבנית עוצרת יצירה עתידית ומשאירה את העבר.
- **פרופיל:** שם מלא, טלפון, מיקום עבודה. מיקום העבודה מוצג על שורות משכורת.
- **יתרת עו"ש:** הכנסות − הוצאות − חיסכון.

## 4 עמודים

### 1. בית — `/`
Hero: ברכה (שם פרטי) + "נשאר בעו״ש החודש" (count-up) + פס ניצול. מתחת: 3 כרטיסי סיכום, עסקאות אחרונות (קטגוריה ככותרת, מהות+תאריך מתחת, תג בעלים צבעוני, לחיצה = עריכה). FAB ➕ בצד — בכל העמודים.

### 2. החודש — `/month?year&month`
Hero: חודש + חצי ניווט + 4 KPI צ'יפים. גוף: התראות חריגה (קטגוריה 50%+ מעל ממוצע 3 חודשים, פס צבע צדדי) · "לאן הלך הכסף" (דונאט) · הוצאות לפי קטגוריה (דונאט + מקרא לחיץ שנפתח לפירוט כל עסקאות הקטגוריה) · חיסכון לפי יעד · הכנסות לפי מקור · הוצאות לפי בן משפחה.

### 3. השוואה — `/months`
Hero: כותרת + סיכומי-כל. גוף: עמודות הכנסות/הוצאות/חיסכון (6 חודשים) · חיסכון מצטבר (קו) · רשימת חודשים → כניסה לעמוד "החודש" של כל חודש.

### 4. הגדרות — `/settings`
Hero: שם מלא + אווטאר. גוף: 4 קבוצות אקורדיון מקופלות:
- **המשפחה** — שם (עריכה), חברים עם פרטים מלאים (עריכה רק לעצמי), העתקת קוד הזמנה
- **קטגוריות** — טאבים לפי סוג; מצב "ניהול" חושף עריכה/מחיקה/הוספה
- **עסקאות קבועות** — רשימה + הסרה
- **אבטחה וחשבון** — שינוי סיסמה (עם סיסמה נוכחית), התנתקות

## מסכי כניסה
- **התחברות/הרשמה** — מסך אחד "ברוך הבא" עם לשוניות. הרשמה: שם פרטי, שם משפחה, אימייל, טלפון (אופציונלי), סיסמה + אימות, קוד הזמנה (אופציונלי).
- **שכחתי סיסמה** — שליחת קישור איפוס למייל → עמוד קביעת סיסמה חדשה.
- **אונבורדינג** (משפחה חדשה בלבד) — 3 שלבים: שם משפחה → קוד הזמנה → בחירת קטגוריות מסט גנרי מוצע.
- **זכור אותי** — סשן 90 יום, רענון טוקן אוטומטי, התנתקות רק מכפתור.

## מודאל הוספה/עריכה (גלובלי, מכל עמוד)
3 כפתורי סוג (הוצאה/הכנסה/חיסכון) → קטגוריות של הסוג בלבד · בורר "של מי?" (הוצאה/הכנסה) · תיאור, תאריך, עסקה קבועה (תדירות + סיום) · במצב עריכה: כפתור מחיקה.
